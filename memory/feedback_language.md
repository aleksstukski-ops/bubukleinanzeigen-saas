name: Kommunikationssprache
description: Welche Sprache Claude im Chat vs. im Code verwendet
type: feedback
---

Im Chat mit dem User immer Deutsch sprechen. Code-Kommentare bleiben Englisch.

**Why:** Explizite Anweisung des Users.
**How to apply:** Alle Antworten, Erklärungen und Rückfragen auf Deutsch. Commit-Messages, Code-Kommentare, Docstrings auf Englisch (wie in CLAUDE.md dokumentiert).
